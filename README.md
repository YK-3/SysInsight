\# SysInsight



AI-Powered System Monitoring and Resource Prediction Platform



\## Overview



SysInsight is a Python-based desktop application developed for real-time system monitoring, historical data analysis, graphical visualization, and machine learning-based resource prediction.



\## Features



\* Real-Time CPU Monitoring

\* RAM Monitoring

\* Disk Monitoring

\* Network Monitoring

\* Health Score Calculation

\* SQLite Database Logging

\* Trend Graph Generation

\* Top Process Analysis

\* AI-Based Resource Prediction

\* Modern Dashboard Interface



\## Technologies Used



\* Python

\* CustomTkinter

\* SQLite

\* Psutil

\* Pandas

\* NumPy

\* Matplotlib

\* Scikit-Learn
